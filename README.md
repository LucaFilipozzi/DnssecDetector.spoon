# DnssecDetector.spoon

This is a replacement for [dnssec-trigger][1] that leverages [hammerspoon][2].

The solution makes use of [MacPorts][3]' unbound, knot (kdig), curl-ca-bundle, curl (+ares +http2 +ssl), and bash.
